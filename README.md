# Smart Task Manager

## Project Overview

Smart Task Manager is a lightweight web application developed as part of the EPAM EliteA Capstone Project.

The project demonstrates an AI-Assisted Software Development Life Cycle (SDLC) using multiple AI Assistants with Human-in-the-Loop (HITL) validation across every SDLC phase.

## Current Features

- Create Task
- View Tasks

## Planned Enhancements

- User Authentication
- Task Priority
- Due Date
- Search Tasks
- Dashboard
- Edit Task
- Delete Task
